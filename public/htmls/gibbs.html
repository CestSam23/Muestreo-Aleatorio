Muestreo de Gibbs

El muestreo de Gibbs es un algoritmo MCMC (Markov Chain Monte Carlo) para generar muestras de una distribución multivariante.

Xᵢ(t+1) ~ P(Xᵢ | X₁(t+1), ..., Xᵢ₋₁(t+1), Xᵢ₊₁(t), ..., Xₙ(t))
Actualización iterativa de cada variable condicionada a las demás

Proceso: Se muestrea cada variable condicionadamente a las demás, iterativamente

Ventaja: Permite muestrear de distribuciones complejas multivariantes

Aplicaciones: Inferencia Bayesiana, procesamiento de imágenes, análisis de sistemas complejos, predicción climática.
