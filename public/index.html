Binomial

Simulación de un muestreo Binomial.

Generamos k números aleatorios x1,x2,..,xk y repetimos el experimento n número de veces
1ra repeticion
x1 → E
x2 → F
...
xk → F

2da repeticion
x1 → F
x2 → E
...
xk → F

"n"va repeticion
x1 → F
x2 → F
...
xk → E
La función de densidad nos muestra los éxitos en los n experimentos.

Exponencial

Simulación de un experimento de función Exponencial.

Generamos n números aleatorios u1,u2,..,un. Una muestra aleatoria con parametro λ con distribución exponencial esta dada por:
x1 = -log(1-u1)/λ
x2 = -log(1-u2)/λ
...
xn = -log(1-un)/λ

Densidad Normal Estandar

Simulación de un muestreo de densidad normal estandar.

Se generan dos muestras de números aleatorios {u1,u2,...,un} y {v1,v2,...,vn}
La muestra aleatoria con distribución normal estandar esta dada por:
x1 = sqrt(-2*log(1/u1))*cos(2π*v1)
x2 = sqrt(-2*log(1/u2))*cos(2π*v2)
...
xn = sqrt(-2*log(1/un))*cos(2π*vn)
De esta forma, {x1,x2,...,xn} se obtiene una muestra aleatoria con distribución normal estandar.

Densidad Normal con media μ y varianza σ²

Simulación de un muestreo de densidad normal con media y varianza especificadas.

Dada una muestra generada por densidad normal estandar, la muestra con media μ y varianza σ² se obtiene mediante la transformación:

y1 = μ + σ * x1

y2 = μ + σ * x2

...

yn = μ + σ * xn
